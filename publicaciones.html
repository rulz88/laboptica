Interfaz gráfica





Hoy en día los sistemas de aplicación de escritorio también conocido como un recurso o un "app", es un programa informático diseñado para ayudar al usuario a realizar singulares o múltiples tareas específicas relacionadas Permite visualizar el comportamiento en tiempo real de los sensores (termopares tipo k). Lo que permitirá conocer el comportamiento del concentrador solar.

MAX6675




Para obtener la información térmica arrojada por los termopares fue necesario realizar el proceso de diseño del circuito para devastado del chip MAX6675. El cual realiza la compensación de unión fría y digitaliza la señal del termopar tipo K. El integrado MAX6675 es capaz de resolver temperaturas desde 0.25 °C y permite lecturas de hasta +1024 °C, y exhibe una exactitud de 8 LSBs para temperaturas que van desde 0 °C a +700 °C. Para este proyecto se empleó la presentación de encapsulado SO de este circuito.

Plataforma Arduino





Se ocupa para enviar los datos a un servidor, basada en el microcontrolador ATmega328, cuenta con una conexión RJ45 y funcionalidad Ethernet. La programación de la placa Arduino abarca desde la inclusión de las librerías del circuito MAX6675 y la API de transmisión de Plotly hasta la lógica de configuración y conexión Ethernet y las peticiones HTTP para el almacenamiento los datos.

Base de datos






Para el almacenamiento de los datos se usó un servidor web donde se almacenó un programa escrito en el lenguaje del lado del servidor PHP para realizar consultas de inserción a la base de datos MySQL. La placa Arduino envía parámetros (valores de los sensores) a dicho programa a través de una petición HTTP.

Graficación (Plotly)



Para llevar a cabo la presentación grafica de los datos, la placa Arduino envía la información a través de dos peticiones HTTP una de las cuales realiza una transmisión vía internet al servicio Plotly y la otra genera una consulta de inserción de los mismos a una base de datos. Plotly es una herramienta de análisis en línea y visualización de datos que proporciona gráficos en línea, análisis, una línea de comandos de Python, y herramientas estadísticas para personas únicas o en colaboración, así como las bibliotecas de graficación científica para Python, R, MATLAB, Perl, Julia, Arduino, y REST.
